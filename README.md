# react-native-wake-on-lan

React Native Wake-on-LAN lib

## Installation

```sh
npm install react-native-wake-on-lan
```

## Usage

```js
import WakeOnLan from "react-native-wake-on-lan";

// ...

const result = await WakeOnLan.multiply(3, 7);
```
